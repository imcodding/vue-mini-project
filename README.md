# vue-mini-project
- 간단한 입력 폼 테스트
- git을 이용하여 mac 과 window 개발환경에서 테스트 진행(vue-cli-service 에러 해결)
### Version
- npm 6.14.8
- vue@2.7.4

### Project setup
Delete 'package-lock.json' file and then execute. If you don't delete the file, there will be an error.
```
npm cache clean --force #npm
npm install
```

### Project start
```
npm run serve
```
